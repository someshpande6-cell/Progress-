# 🛠️ SolidWorks Daily Practice

Hi! I'm a student currently learning SolidWorks and improving my CAD design skills step by step.

This repository is my daily practice log where I upload **one 3D part every day**. Each model is created based on given dimensions or design exercises to strengthen my understanding of:

- Sketching
- Feature tools (Extrude, Revolve, Sweep, etc.)
- Dimensioning & constraints
- Design thinking

## 📅 Goal
To stay consistent and improve my SolidWorks skills by practicing daily and documenting my progress.

## 📂 What you'll find here
- Daily SolidWorks part files (.SLDPRT)
- Screenshots of models
- Step-by-step modeling (when possible)
- Dimensions used in each design

## 📈 Why this repo?
I believe consistency is the key to mastering CAD design. This repository helps me:
- Track my learning journey
- Build a strong portfolio
- Share my work with others
