Step 1: Start a New Sketch
•	Open SolidWorks → New Part 
•	Select Front Plane 
•	Click Sketch 
________________________________________
Step 2: Draw Base Profile (Outer Shape)
•	Use the Line tool to create the outer stepped profile. 
•	Start from the bottom-left corner (origin). 
Add dimensions:
•	Total height (left side): 250 mm 
•	Bottom horizontal segments: 
o	100 mm 
o	80 mm 
o	150 mm 
o	150 mm 
•	Right vertical height: 150 mm 
•	Upper right horizontal: 75 mm 
•	Top middle horizontal: 115 mm 
•	Step height (top left cut): 50 mm 
•	Inner vertical step (right): 100 mm 
________________________________________
Step 3: Create Bottom Cut Section
•	From the bottom profile: 
o	Move 100 mm → go up 50 mm → move right 80 mm → go down back to base 
•	This creates the rectangular notch 
________________________________________
Step 4: Create Upper Step (Left Side)
•	From the top left: 
o	Move right 115 mm 
o	Move down 50 mm 
•	Connect back to main profile 
________________________________________
Step 5: Fully Define Sketch
•	Use Smart Dimension tool 
•	Apply all given dimensions: 
o	Horizontal lengths 
o	Vertical heights 
•	Ensure sketch status shows: Fully Defined 
________________________________________
Step 6: Draw Central Circle
•	Select Circle tool 
•	Place circle roughly in center 
Add dimensions:
•	Diameter: Ø80 mm 
•	Vertical position from top: 80 mm 
________________________________________
Step 7: Draw Right Circle
•	Create another circle on right section 
Add dimensions:
•	Diameter: Ø60 mm 
•	Horizontal distance from right edge: 75 mm 
•	Vertical distance from top: 75 mm 
________________________________________
Step 8: Final Constraints
•	Ensure: 
o	Circles are properly constrained 
o	All entities are black (fully defined) 
________________________________________
 Final Step (Optional for 3D)
If required:
•	Go to Features → Extruded Boss/Base 
•	Apply thickness (e.g., 10 mm or as given) 
________________________________________
 Summary
•	The model is built using a single fully defined sketch 
•	Key techniques used: 
o	Stepped profile using lines 
o	Smart dimensioning 
o	Two circular features (Ø80 and Ø60) 
•	The design follows a parametric approach, ensuring accuracy and easy modification
