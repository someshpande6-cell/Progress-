Step 1: Sketch (Base Layout)
Start a new part and select the Top Plane
Create a vertical layout using a centerline to align all pulleys
This acts as the reference axis for symmetry
Dimensions used: Vertical spacing: 50 mm (top to center) and 50 mm (center to bottom)
Plane: Top Plane
Step 2: Sketch (Top Pulley)
Draw two concentric circles at the top position
Dimensions used:
Outer diameter: Ø30 mm
Inner diameter: Ø14 mm
Ensure center lies on the vertical centerline
Plane: Top Plane
Step 3: Sketch (Center Pulley)
Draw two concentric circles at the center position
Dimensions used:
Outer diameter: Ø60 mm
Inner diameter: Ø40 mm
Align center with vertical reference line
Plane: Top Plane
Step 4: Sketch (Bottom Pulley)
Draw two concentric circles at the bottom position
(Same as top pulley)
Dimensions used:
Outer diameter: Ø30 mm
Inner diameter: Ø14 mm
Maintain vertical alignment
Plane: Top Plane
Step 5: Sketch (Tangent Belt – Left Side)
Use Line tool to draw a slanted line connecting:
Top pulley outer circle
Center pulley outer circle
Bottom pulley outer circle
Apply Tangent relations to all three circles
Dimensions used: Controlled by tangency (no direct length given)
Plane: Top Plane
Step 6: Sketch (Tangent Belt – Right Side)
Repeat the same on the right side:
Draw slanted line connecting all three pulleys
Apply Tangent relations with all circles
Ensure symmetry with left side
Plane: Top Plane
Step 7: Trim (Profile Cleanup)
Use Trim Entities
Remove unnecessary inner arc portions where belt overlaps circles
Keep only the outer visible belt profile
Step 8: Fully Define Sketch
Apply all remaining constraints:
Concentric relations for circles
Vertical alignment of centers
Tangent relations for belt lines
Ensure sketch status shows: Fully Defined
Step 9: Extruded Boss/Base (3D Creation – Optional)
Select the closed profile
Go to Features → Extruded Boss/Base
Apply thickness (e.g., 10–20 mm)
🔍 Final Summary
The model is created using a single 2D sketch consisting of:
Three pulleys (top, center, bottom)
Concentric circles for each pulley
Tangent lines forming a belt path
Key techniques used:
Tangent constraints
Concentric relations
Symmetry using centerline
The approach ensures a fully parametric and editable design
