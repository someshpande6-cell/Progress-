Step 1: Start Sketch
Open SolidWorks → New Part
Select Top Plane
Click Sketch
Step 2: Draw Centerline (Reference)
Use Centerline tool
Draw a horizontal line from left to right
This will act as a reference axis for both circles
Step 3: Draw Left Circle
Select Circle tool
Draw a circle on the left side
Apply dimension:
Diameter: Ø60 mm
Step 4: Draw Right Circle
Draw another circle on the right side along the same centerline
Apply dimensions:
Diameter: Ø40 mm
Distance between centers: 100 mm
Step 5: Align Circles
Select both circle centers
Make them Horizontal (collinear)
Ensure both lie on the centerline
Step 6: Draw Connecting Lines
Use Line tool
Draw:
One line connecting top edge of left circle to top edge of right circle
One line connecting bottom edge of left circle to bottom edge of right circle
Step 7: Apply Tangent Relations (IMPORTANT)
Select:
Top line + left circle → Tangent
Top line + right circle → Tangent
Bottom line + left circle → Tangent
Bottom line + right circle → Tangent

👉 This is the key step that creates the smooth transition

Step 8: Trim Extra Portions
Use Trim Entities (Power Trim)
Remove unwanted parts of circles inside the shape
Keep only the outer profile
Step 9: Fully Define Sketch
Ensure:
All dimensions are applied
Sketch turns black (Fully Defined)
Step 10: (Optional) Convert to 3D
Go to Features → Extruded Boss/Base
Apply thickness (e.g., 10 mm)
