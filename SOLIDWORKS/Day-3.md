Step 1: Start Sketch
Open SolidWorks → New Part
Select Front Plane
Click Sketch
Step 2: Draw Center Geometry (Circles)
Use Circle tool
Draw two concentric circles at the origin
Apply dimensions:
Inner circle: Ø80 mm
Outer circle: Ø120 mm

👉 These form the central hub

Step 3: Create Reference Lines (Construction)
Draw horizontal and vertical centerlines
These help maintain symmetry in all directions
Step 4: Draw One Arm (Base Shape)
Focus on creating one side first (e.g., bottom arm)
Steps:
Draw a horizontal line: 75 mm
From its end, draw an inclined line at 45°
Continue upward to meet near the outer circle region
Add vertical segment: 75 mm
Connect back toward center

👉 This forms one “arm” of the X-shape

Step 5: Apply Dimensions to Arm
Horizontal length: 75 mm
Vertical height: 75 mm
Inclined angle: 45°
Slanted edge length (right side): 135 mm
Step 6: Trim with Circle
Use Trim Entities
Trim arm edges where they intersect the outer circle (Ø120)
Keep a clean intersection
Step 7: Create Full X-Shape Using Mirror
Select all entities of the first arm
Use Mirror Entities
Mirror 1:
Mirror about vertical centerline → creates left-right symmetry
Mirror 2:
Mirror about horizontal centerline → creates full X shape
Step 8: Clean Up Geometry
Trim extra overlapping lines
Ensure smooth intersections between arms and circle
Step 9: Apply Remaining Dimensions
Outer arm length: 135 mm
Ensure all angles are 45°
Verify symmetry
Step 10: Fully Define Sketch
Ensure:
All lines are constrained
Sketch becomes Fully Defined (Black)
Step 11: (Optional) Convert to 3D
Go to Features → Extruded Boss/Base
Apply thickness (e.g., 10–20 mm)
🔍 Summary
Built using:
Two concentric circles (Ø80 & Ø120)
One arm + mirror (efficient method)
45° angled geometry
Key concepts:
👉 Symmetry
👉 Mirror entities
👉 Trim with circular boundaries
